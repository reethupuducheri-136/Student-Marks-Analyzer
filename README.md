# 🎓 Student Marks Analyzer

A beginner-friendly Python project that analyzes student marks and generates a complete performance report.

---

## 🚀 Features

✅ Accepts student name and subject marks  
✅ Calculates total marks  
✅ Calculates average marks  
✅ Displays grade based on performance  
✅ Finds highest and lowest marks  
✅ Shows subject-wise pass/fail status  
✅ Clean and easy-to-understand output
